# Resume Placeholder

## Instructions

1. Add your resume PDF file to this directory
2. Name it exactly: `Suganraj_S_Resume.pdf`
3. The "View Resume" button in the Hero section will link to this file

## File Location

The resume should be placed at:
```
public/resume/Suganraj_S_Resume.pdf
```

## Alternative

If you prefer a different filename, update the `resume` field in `src/data/contact.js`:

```javascript
resume: '/resume/Your_Filename.pdf',
```
