# Placeholder Images

This directory contains placeholder images that should be replaced with actual assets:

## Required Replacements

1. **profile-placeholder.svg** → Replace with your actual profile photo
   - Recommended: Square image (800x800px or larger)
   - Format: JPG or PNG
   - Update the `src` in `src/components/Hero.jsx` if using a different filename

2. **favicon.svg** → Replace with your custom favicon
   - Recommended: 32x32px or 64x64px
   - Format: ICO, PNG, or SVG
   - Also update `index.html` if using a different filename

3. **og-image.svg** → Replace with Open Graph image for social sharing
   - Recommended: 1200x630px
   - Format: PNG or JPG
   - This appears when sharing the site on social media

## Quick Setup

1. Add your profile photo as `profile-placeholder.svg` (or update the filename in Hero.jsx)
2. Add your favicon as `favicon.ico` or `favicon.svg`
3. Create an og-image.png (1200x630px) for social sharing
